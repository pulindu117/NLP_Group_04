# Models Folder

This folder contains generated model artifacts (vectorisers, 
tokenisers, trained models, feature matrices). These files are 
regenerated by running the notebooks in order and are excluded 
from version control due to their large file size.

To regenerate:
1. Run notebooks/01_preprocessing_pulindu_pasanjith.ipynb
2. Run notebooks/03_features_pulindu_pasanjith.ipynb
3. Run notebooks/04_models_pulindu_pasanjith.ipynb
